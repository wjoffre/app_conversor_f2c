# flutter_conv_f_to_c

Este é um App que propus aos meus alunos da disciplina de programação mobile, onde usamos o framework Flutter.

A intenção é que o App receba uma entrada de dados em graus Farenheit e aplique a conversão para graus Celsius. 

Nesta versão ainda não estou usando o teclado para fazer a entrada de dados, mas sim um botão para incrementar, outro para decrementar e um para fazer a conversão.
